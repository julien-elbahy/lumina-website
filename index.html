Frequently asked questions

Got questions?

Is Lumina really free?+
Yes. Lumina is 100% free with no account required. Most features work instantly without any setup. Some advanced features (keyword research, AI content tools) use your own API keys from third-party providers — you only pay what you use directly to them.
What is GEO (Generative Engine Optimization)?+
GEO is the practice of optimizing your content for AI-powered search engines like ChatGPT, Perplexity, Claude, and Google's AI Overviews. Lumina helps you check which AI crawlers can access your site, verify your llms.txt file, and track AI-driven traffic through your GA4 data.
Does Lumina collect my data?+
No. All analysis happens locally in your browser. Lumina has no backend server, no analytics, and no tracking. API keys and OAuth tokens are stored in Chrome's local storage on your device. Read the full privacy policy for details.
Do I need API keys to use Lumina?+
No. Over 90% of features work without any keys — on-page audits, heading analysis, link checking, schema validation, robots.txt parsing, llms.txt checking, hreflang validation, JS vs No-JS, screenshots, and more. API keys unlock keyword research (DataForSEO), AI content tools (OpenAI), and live GSC/GA4 data (free Google OAuth).
What permissions does Lumina need and why?+
Lumina needs activeTab and tabs to read the current page, webRequest for HTTP headers, scripting for on-page features like heading highlighting, and host_permissions for robots.txt fetching and API calls. All permissions are explained in the privacy policy.
