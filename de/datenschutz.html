Datenschutzerklärung

Letzte Aktualisierung: 23. März 2026

Lumina – SEO & GEO ("Lumina", "die Extension") ist eine Chrome-Browser-Extension, erstellt von Julien El-Bahy. Diese Datenschutzerklärung erklärt, wie die Extension mit Ihren Daten umgeht.

Datenerhebung

Lumina erhebt, überträgt oder speichert keine personenbezogenen Daten auf externen Servern. Die Extension verwendet keine Analyse-Tools, Tracking-Skripte oder Werbung jeglicher Art.

Wie Ihre Daten verarbeitet werden

Alle SEO-Analysen (Seiten-Audits, Heading-Checks, Link-Analysen, Schema-Validierung etc.) werden lokal in Ihrem Browser durchgeführt. Die Extension liest den Inhalt des aktuell aktiven Tabs für die Analyse. Diese Daten werden niemals an einen Server gesendet, der von Lumina oder Dritten betrieben wird.

API-Schlüssel

Lumina ermöglicht optional die Verbindung eigener API-Schlüssel für erweiterte Funktionen:

  • OpenAI API-Schlüssel: Für KI-gestützte Content-Tools. Anfragen werden direkt von Ihrem Browser an die OpenAI-API gesendet.
  • DataForSEO API-Schlüssel: Für Keyword-Recherche und SERP-Analyse. Anfragen gehen direkt an die DataForSEO-API.
  • Google PageSpeed Insights API-Schlüssel: Für Performance-Analysen. Anfragen gehen direkt an Googles API.

Alle API-Schlüssel werden im lokalen Speicher von Chrome (chrome.storage.local) auf Ihrem Gerät gespeichert. Sie werden niemals an andere Server als den jeweiligen API-Anbieter übertragen.

Google OAuth (Search Console & Analytics)

Lumina ermöglicht die Verbindung Ihrer Google Search Console und Google Analytics 4 Konten über Googles OAuth 2.0 Autorisierung. Dabei gilt:

  • Sie werden zu Googles Consent-Screen weitergeleitet, wo Sie Zugriff auf webmasters.readonly (Search Console) und/oder analytics.readonly (Analytics) Scopes gewähren.
  • Access-Tokens werden in Chrome's Session Storage gespeichert und beim Schließen des Browsers gelöscht.
  • Refresh-Tokens werden in Chrome's Local Storage für dauerhaften Zugriff gespeichert.
  • Alle Datenanfragen gehen direkt von Ihrem Browser zu Googles APIs. Lumina betreibt keinen Backend-Server.
  • Sie können die Verbindung jederzeit über die Einstellungen der Extension trennen.

Lumina fordert ausschließlich Lesezugriff an. Es kann keine Daten in Ihrer Search Console oder Analytics ändern.

Lokal gespeicherte Daten

Die Extension speichert folgende Daten lokal auf Ihrem Gerät:

  • Ihre Einstellungen und Präferenzen (Theme, API-Keys, Standardoptionen)
  • Zwischengespeicherte Keyword-Daten (um redundante API-Aufrufe zu reduzieren)
  • Google OAuth Refresh-Tokens (wenn Sie GSC/GA4 verbinden)

Sie können alle gespeicherten Daten jederzeit über die Einstellungen der Extension löschen ("Alle Daten zurücksetzen").

Berechtigungen

Die Extension benötigt folgende Chrome-Berechtigungen:

  • storage — Einstellungen und API-Keys lokal speichern
  • activeTab / tabs — Aktuelle Seite für Analysen lesen
  • scripting — Content-Scripts für On-Page-Features (Heading-Highlighting, Link-Checker)
  • webRequest — HTTP-Response-Headers erfassen (Statuscodes, X-Robots-Tag)
  • host_permissions (<all_urls>) — Für robots.txt-Abruf, Link-Checks und API-Aufrufe
  • identity — Google OAuth Authentifizierung
  • clipboardWrite — Text/Daten in die Zwischenablage kopieren
  • downloads — Screenshots und CSV-Dateien exportieren

Drittanbieter-Dienste

Bei Nutzung API-verbundener Features werden Daten an folgende Drittanbieter gesendet:

  • Google (Search Console, Analytics, PageSpeed)
  • OpenAI (KI-Content-Tools)
  • DataForSEO (Keyword-Recherche, SERP-Daten)

Datenschutz von Kindern

Lumina richtet sich nicht an Kinder unter 13 Jahren. Wir erheben wissentlich keine Daten von Kindern.
