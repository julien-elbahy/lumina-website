Häufige Fragen

Welche Schema-Typen werden unterstützt?+
Article, NewsArticle, BlogPosting, Product, FAQPage, LocalBusiness, Organization, Person, BreadcrumbList, Event, Recipe, HowTo, VideoObject, WebSite und Review — jeweils mit typspezifischen Pflicht- und empfohlenen Feldern.
Wie unterscheidet sich das von Googles Rich Results Test?+
Googles Tool prüft Rich-Result-Berechtigung. Unseres fokussiert auf Feld-Level-Validierung — Pflichtfelder, Datumsformate, URL-Gültigkeit, verschachtelte Objekte. Beide ergänzen sich.
Kann ich eine Live-URL validieren?+
Ja! Gib eine URL in die "Laden & Validieren" Leiste ein. Das Tool ruft die Seite ab, extrahiert alle JSON-LD Blöcke und validiert jeden einzelnen.
