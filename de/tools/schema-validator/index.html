Schema JSON-LD Validator

This tool parses JSON-LD and checks against Google's documentation — required fields for 15 schema types, ISO 8601 dates, URL formats, and nested objects.

Welche Schema-Typen werden unterstützt?+
Article, NewsArticle, BlogPosting, Product, FAQPage, LocalBusiness, Organization, Person, BreadcrumbList, Event, Recipe, HowTo, VideoObject, WebSite und Review.
Kann ich von einer Live-URL validieren?+
Ja! Verwende "Fetch & Validate", um alle JSON-LD-Blöcke von einer beliebigen URL abzurufen und zu validieren.
