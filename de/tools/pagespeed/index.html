FAQ

Ist das kostenlos?+
Ja. Die Google PageSpeed Insights API ist kostenlos und braucht keinen API-Key. Das Tool ruft sie direkt aus deinem Browser auf.
Warum unterscheidet sich mein Score von Chrome DevTools?+
PageSpeed Insights führt Lighthouse auf Googles Servern aus, während DevTools es auf deinem Rechner ausführt. Netzwerkbedingungen und Hardware unterscheiden sich, was zu leicht unterschiedlichen Ergebnissen führt.
Was ist ein guter PageSpeed-Score?+
90–100 gilt als gut, 50–89 verbesserungswürdig und unter 50 als schlecht. Konzentriere dich darauf, 90+ für Mobilgeräte zu erreichen, da Google Mobile-First-Indexierung für Rankings verwendet.
Was sind Core Web Vitals?+
Core Web Vitals sind drei Metriken, die Google als Ranking-Signale nutzt: LCP (Largest Contentful Paint) misst die Ladegeschwindigkeit, INP (Interaction to Next Paint) misst die Reaktionsfähigkeit und CLS (Cumulative Layout Shift) misst die visuelle Stabilität.
Wie unterscheidet sich das von Chrome DevTools?+
Dieses Tool nutzt die PageSpeed Insights API, die reale Felddaten aus dem Chrome UX Report mit Labordaten von Lighthouse kombiniert. DevTools liefert nur Labordaten basierend auf deinem lokalen Rechner und Netzwerk.
