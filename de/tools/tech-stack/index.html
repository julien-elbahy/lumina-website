FAQ

Wie funktioniert die Erkennung?+
Das Tool ruft den HTML-Quellcode der Seite ab und sucht nach bekannten Script-URLs, globalen Variablen und Code-Mustern, die für jede Technologie spezifisch sind. GA4 wird beispielsweise durch das Auffinden von gtag( oder googletagmanager.com/gtag im Quellcode erkannt.
Kann das Tool alles erkennen?+
Das Tool erkennt Technologien, die Spuren im HTML-Quellcode hinterlassen. Serverseitige Tools, Backend-Integrationen oder Scripts, die von einer eigenen Domain geladen werden, können möglicherweise nicht erkannt werden. Die Lumina-Extension erkennt mehr, da sie im Browser nach der JavaScript-Ausführung läuft.
Wird das Meta-Tag „generator" verwendet?+
Ja! Das Tool prüft auch das <meta name="generator">-Tag, das viele CMS-Plattformen automatisch setzen (WordPress, Joomla, Drupal usw.).
