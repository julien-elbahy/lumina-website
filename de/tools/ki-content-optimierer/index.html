Wie funktioniert die Bewertung?+
Der Content-Score kombiniert Keyword-Dichte (ideal: 1–3%), Lesbarkeit (Flesch-Kincaid), Inhaltslänge (300+ Wörter empfohlen) und Satzvarianz. Höhere Scores zeigen bessere On-Page-SEO-Ausrichtung für dein Zielkeyword.
Ist die KI-Analyse kostenlos?+
Ja — du erhältst 10 kostenlose KI-Abfragen pro Tag (geteilt über alle KI-Features). Für unbegrenzten Zugriff gib deinen eigenen OpenAI API Key ein. Die lokale Analyse (Wortanzahl, Dichte, Lesbarkeit) funktioniert immer ohne Limits.
