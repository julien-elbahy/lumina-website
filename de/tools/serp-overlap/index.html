SERP Overlap Checker

Vergleiche Google-Suchergebnisse für zwei Keywords. Finde Domains, die für beide ranken — entscheidend für das Verständnis von Keyword-Kannibalisierung und Topic-Clustering.

Fetching results for both keywords…

Was ist SERP Overlap?

SERP Overlap misst, wie viele Domains für beide Keywords ranken. Hoher Overlap (>60%) bedeutet, Google betrachtet die Keywords als sehr ähnlich — sie könnten Keyword-Kannibalisierung verursachen. Niedriger Overlap bedeutet, die Keywords bedienen unterschiedliche Suchintentionen und sollten separate Seiten haben.

Warum verwendet das Tool 2 Abfragen?+
Jedes Keyword erfordert eine separate Google-SERP-Abfrage. Das Tool ruft beide ab, dann vergleicht es die Domain-Listen lokal. Mit dem Free-Tier (5/Tag) kannst du 2 Overlap-Vergleiche machen.
Was ist Keyword-Kannibalisierung?+
Keyword-Kannibalisierung tritt auf, wenn mehrere Seiten deiner Website für dasselbe Keyword konkurrieren. Hoher SERP-Overlap zwischen zwei Keywords deutet darauf hin, dass sie dieselbe Suchintention bedienen — separate Seiten können dein Ranking-Potenzial aufteilen.
Wie funktioniert der Overlap-Score?+
Der Overlap-Score ist der Prozentsatz der Domains, die in den Top-Ergebnissen für beide Keywords erscheinen. Ein Score über 60 % zeigt starke Ähnlichkeit in der Suchintention, unter 30 % deutet auf unterschiedliche Intentionen hin.
Wann sollte ich Inhalte zusammenführen?+
Erwäge das Zusammenführen von Seiten, wenn der Overlap über 60 % liegt und keine der Seiten einzeln gut rankt. Die Konsolidierung in eine umfassende Seite kann Ranking-Signale bündeln und die Performance für beide Keywords verbessern.
Kann ich mehr als zwei Keywords vergleichen?+
Das Tool vergleicht jeweils zwei Keywords. Für eine breitere Analyse kannst du mehrere Vergleiche durchführen, um Cluster verwandter Keywords zu identifizieren und herauszufinden, welche die gleiche Suchintention teilen.
