How it works

This tool fetches the page with full JavaScript rendering, then extracts the visible text content (excluding scripts, styles, and hidden elements). Words are counted by splitting on whitespace, sentences by periods/question marks/exclamation marks.

FAQ

What does this tool count?+
It counts words, characters, sentences, and paragraphs from the visible text content of any page. Scripts, styles, and hidden elements are excluded to give you an accurate content-only count.
What is a good word count for SEO?+
There is no universal ideal, but most top-ranking pages have 800-2,000 words for informational content. Focus on covering the topic thoroughly rather than hitting a specific number.
How is reading time calculated?+
Reading time is estimated using an average reading speed of 200-250 words per minute. The total word count is divided by this rate to give an approximate reading duration.
Does word count affect rankings?+
Word count itself is not a ranking factor. However, longer content tends to cover topics more comprehensively, which can satisfy search intent better and earn more backlinks.
Can I analyze any page?+
Yes, enter any public URL and the tool will fetch and analyze its content. The page is rendered with JavaScript enabled so dynamically loaded content is included in the count.
