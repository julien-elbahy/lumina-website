FAQ

Is this free?+
Yes. Google's PageSpeed Insights API is free and requires no API key. This tool calls it directly from your browser.
Why is my score different than in Chrome DevTools?+
PageSpeed Insights runs Lighthouse on Google's servers, while DevTools runs it on your machine. Network conditions and hardware differ, leading to slightly different results.
What is a good PageSpeed score?+
90-100 is considered good, 50-89 needs improvement, and below 50 is poor. Focus on achieving 90+ for mobile, as Google uses mobile-first indexing for ranking.
What are Core Web Vitals?+
Core Web Vitals are three metrics Google uses as ranking signals: LCP (Largest Contentful Paint) measures loading speed, INP (Interaction to Next Paint) measures responsiveness, and CLS (Cumulative Layout Shift) measures visual stability.
How does this differ from Chrome DevTools?+
This tool uses the PageSpeed Insights API which combines real-world field data from the Chrome UX Report with lab data from Lighthouse. DevTools only provides lab data based on your local machine and network.
