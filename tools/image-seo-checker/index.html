Why image SEO matters

Images often make up over half of a page's total weight. Missing ALT text hurts both accessibility and search visibility. Modern formats like WebP and AVIF cut file sizes significantly compared to JPEG or PNG. Missing width and height attributes cause Cumulative Layout Shift (CLS), a Core Web Vital. This tool checks all of these factors and gives you an actionable score.

FAQ

What does this tool check?+
It analyzes all images on a page for ALT tags, file format (WebP, AVIF, PNG, JPEG), lazy loading attributes, and file size to identify SEO and performance issues.
Why are ALT tags important for SEO?+
ALT tags help search engines understand image content and are essential for accessibility. They also allow your images to appear in Google Image Search results, driving additional traffic.
What image formats are best for web?+
WebP and AVIF offer the best compression, reducing file sizes by 25-50% compared to JPEG and PNG while maintaining visual quality. Most modern browsers support both formats.
Does this tool check lazy loading?+
Yes, it detects whether images use the native loading="lazy" attribute. Lazy loading defers off-screen images, improving initial page load speed and Core Web Vitals scores.
What is the ideal image file size?+
Aim for under 100 KB per image. Hero images can be up to 200 KB, but smaller is always better for page speed. Use modern formats and proper compression to stay within these limits.
