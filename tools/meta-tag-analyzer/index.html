Frequently asked questions

What meta tags does this tool check?+
Title tag (with character count), meta description (with character count), canonical URL, robots meta tag, H1 tags, all Open Graph tags (og:title, og:description, og:image, og:url, og:type), Twitter Card tags, Schema.org JSON-LD types, hreflang tags, viewport, charset, lang attribute, and favicon.
Can I analyze any website?+
Yes — any publicly accessible URL. Some sites behind login walls or with strict bot protection may not be fetchable. For those, the Lumina Chrome Extension works directly in your browser without any proxy.
Is this tool free?+
Completely free, no signup needed. The analysis runs through our own server — no third-party proxies. Your data stays private.
