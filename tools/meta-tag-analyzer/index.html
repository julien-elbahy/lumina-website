How this tool works

Enter any public URL and this tool fetches the page through our own servers — no browser extension needed. It parses the full HTML and extracts all SEO-relevant meta tags: title, description, canonical, robots, Open Graph, Twitter Cards, Schema.org JSON-LD, hreflang, viewport, charset, and more.

Results are organized in 6 tabs covering on-page SEO (title, description, H1, canonical, robots), heading counts, Open Graph tags, Twitter Card tags, Schema.org detection, and technical setup (hreflang, viewport, charset, lang, favicon). A weighted SEO score (0–100) gives you an instant overview of how well-optimized the page is, with actionable tips for every issue found.

For JavaScript-heavy sites, the tool automatically falls back to a headless browser renderer to ensure accurate results — just like a search engine would see the page.

This tool is strictly a diagnostic audit. It tells you what's on the page, what's missing and what's broken. When you want to rewrite your title and description with live Google-snippet preview, use the SERP Preview editor. When you want visual social card previews, use the OG Preview. Each specialist tool goes deeper than a generic all-in-one audit could.

FAQ

What meta tags does this tool check?+
Title, description, canonical, robots, H1, all Open Graph tags (og:title, og:description, og:image, og:url, og:type, og:site_name, og:locale), Twitter Card tags, Schema.org JSON-LD, hreflang, viewport, charset, lang attribute, and favicon.
Is this tool free?+
Completely free, no signup needed. The analysis runs through our own servers — no third-party proxies. You can analyze unlimited URLs.
How is the SEO score calculated?+
The score starts at 100 and uses tier-based penalties. Critical failures (missing title/description, noindex, HTTP errors) automatically cap the score at 15–40 regardless of other checks. Important issues (missing H1, bad canonical, broken OG tags, pixel-width overflow) subtract 4–10 points each. Nice-to-have issues (viewport, lang, charset, schema) subtract 2–3 points. Duplicate tags and meta-tag conflicts from the Issues Banner each subtract 2–6 more points. The scoring is also page-type aware: homepages are allowed short brand titles, articles are checked for article-specific signals. A score of 80+ means the meta setup is solid; 50–79 needs improvement; below 50 has critical issues.
Does this tool work with JavaScript-heavy websites?+
Yes. If the initial fetch returns incomplete data (common with React, Angular, or Vue apps), the tool automatically falls back to a headless browser that renders JavaScript — so you see the page exactly as search engines do.
Can I rewrite my meta tags with AI here?+
This tool is strictly a diagnostic analyzer — it tells you what's on the page, what's broken, and what's missing. For rewriting, use the SERP Preview tool: it has a live Google snippet editor with desktop/mobile previews, keyword highlighting and AI-powered title & description suggestions. After every analysis we show an "Open in SERP Preview" button that passes the URL directly.
Why do pixel widths matter for titles and descriptions?+
Google truncates search snippets based on pixel width, not character count. A title with wide characters like "W" or "M" gets cut sooner than one with narrow characters. This tool measures actual rendered pixels for more accurate truncation predictions than character counting alone.
Can I analyze pages behind a login?+
No, this tool can only analyze publicly accessible pages. For pages behind authentication, use the Lumina Chrome extension which runs directly in your browser and can analyze any page you have access to.
